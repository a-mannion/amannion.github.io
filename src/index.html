Masters thesis project in machine learning & biomedical text mining, the goal of which was to develop predictive algorithms to help GPs working with SOS Médecins to more effectively treat patients.

  • Implemented a custom data processing pipeline for patient data from the mobile application used by these GPs, identifying clinically relevant information in order to build a dataset of relevant and usable electronic health records.
  • Designed an experimental paradigm for the comparison and validation of various machine learning methods to make inferences from this data (physiological measurements, pharmacological and medication-related information, observations and clinical reports).
  • Constructed a machine learning framework capable of predicting patients' likelihood of reaccess to the GP healthcare system and probability of serious illness & hospitalisation with ~95% precision.
